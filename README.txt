LayOver README

--Overview--
LayOver is a SwiftUI app that visualizes data on a map with animated backgrounds. Users can import CSV files to display data points as colored annotations based on severity levels.

--Features--
Map visualization with customizable annotations
Animated gradient backgrounds and sparkle effects
CSV file import for data processing and visualization

--Usage--
Import Data: Select up to 5 CSV files for data visualization.
Set Year: Input a year to filter the data for visualization.
Visualize: Initiate visualization to see data points on the map.

--Requirements--
iOS 14.0+
Xcode 12.0+

--Installation--
Download the source code, open with Xcode, build, and run.
